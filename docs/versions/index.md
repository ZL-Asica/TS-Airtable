# Documentation Versions

The root documentation follows the current repository state. Version snapshots
below are generated from release tags so readers can keep using the docs that
match the package version they installed.

<VersionList />

## Release Channels

- Use the **stable** docs when installing from the npm `latest` tag.
- Use the **prerelease** docs when installing from the npm `beta` tag.
- Check the [changelog](https://github.com/ZL-Asica/TS-Airtable/blob/main/CHANGELOG.md)
  when moving between channels.
